# Employee Management System – Full Stack

This is a **Full Stack Employee Management System** built using **React.js** (Frontend) and **Spring Boot + MongoDB** (Backend).  
The system allows users to manage employee records efficiently with a clean and responsive UI.

---

##  Features

- View all employees in a tabular format
- Add new employee
- Edit existing employee
- Delete an employee
- Responsive and clean UI
- Backend REST APIs with Spring Boot
- MongoDB integration for database
- Fully functional Full Stack application

---

##  Tech Stack

**Frontend:**
- React.js
- JavaScript (ES6+)
- Axios – for API calls
- Bootstrap – UI styling
- HTML5 & CSS3

**Backend:**
- Java 17
- Spring Boot
- Spring Data MongoDB
- MongoDB Database
- RESTful API
- Maven for dependency management

---

##  Backend API Endpoints

| Operation | URL | Method |
|-----------|-----|--------|
| Get all employees | `/Emp` | GET |
| Get employee by ID | `/Emp/{id}` | GET |
| Add employee | `/Emp` | POST |
| Update employee | `/Emp/{id}` | PUT |
| Delete employee | `/Emp/{id}` | DELETE |
